About Me

Migrated to Azure (8/29)

  • Shut down EC2
  • Migrated to Azure Web Apps
  • Implemented CI through Gitlab
Getting hands on with Apache and Ubuntu Server on AWS was fun, but I can't justify paying $3 a month for EC2. Initially I migrated to S3 but soon discovered that it was not appropriate for active projects due to high cost of writing to S3. Instead I have migrated almost everything to Azure Web App containers, which are available for the low price of free, so long as I don't go over the quota. I also took this migration time to finally learn some of Gitlab's CI and set it up for auto deployment.
